# Bitácora de uso de Inteligencia Artificial

Durante el desarrollo del proyecto utilicé inteligencia artificial como apoyo para ordenar el informe, entender algunos conceptos de seguridad y mejorar la presentación del sitio web desarrollado en React.

Las respuestas obtenidas sirvieron como una base para comenzar el trabajo, pero en la mayoría de los casos fue necesario revisar el contenido, hacer cambios y adaptarlo al caso de la Municipalidad de Cerro Verde y a la rúbrica entregada por el profesor.

---

# Organización del informe

**Prompt utilizado**

> Ayúdame a ordenar el informe siguiendo la rúbrica del profesor. Quiero que cada sección responda a los criterios de evaluación y que el lenguaje sea simple.

**Resultado**

La respuesta me ayudó a reorganizar el contenido del informe y a identificar qué información faltaba en cada apartado. Después hice varios cambios para adaptarlo al proyecto y dejar la estructura definitiva.

---

# Desarrollo del sitio en React

**Prompt utilizado**

> Ayúdame a mejorar el diseño de mi proyecto en React. Quiero un menú de navegación, tarjetas para separar el contenido y una interfaz más ordenada.

**Resultado**

La respuesta sirvió como punto de partida para mejorar la interfaz. Después fui modificando el diseño, agregando nuevas secciones y ajustando los componentes para que coincidieran con el contenido del informe.

---

# Redacción de las vulnerabilidades

**Prompt utilizado**

> Explícame SQL Injection, Reflected XSS y Command Injection de forma simple para poder escribir mi informe.

**Resultado**

La IA entregó una explicación general de cada vulnerabilidad. Posteriormente adapté la información al contexto de la Municipalidad de Cerro Verde, agregué las evidencias obtenidas durante la auditoría y utilicé los resultados del análisis CVSS para completar cada sección.

---

# Reflexión

Durante este proyecto la inteligencia artificial fue una herramienta de apoyo para organizar ideas, comprender algunos conceptos y mejorar la presentación del trabajo.

Aun así, fue necesario revisar gran parte de las respuestas, corregir información, adaptar los textos al caso asignado y reorganizar varias secciones para cumplir con la rúbrica. En mi experiencia, la IA ayuda a ahorrar tiempo y a tener una base desde donde empezar, pero el resultado final depende de la revisión y las modificaciones que realiza el propio estudiante.
